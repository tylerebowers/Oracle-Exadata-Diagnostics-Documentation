Net - Exalogic: Network Diagnostics 
  This menu enables you to collect Exalogic network diagnostic 
  data based on a specific issue. Exalogic network components 
  have been grouped into the categories below. 

  Select one of the following options:
    (1) Performance (Network Health)
    (2) Network Cabling
    (3) IB Switch
    (4) IB Traffic
    (5) General

    (M)ain    (B)ack    (H)elp   (E)xit

[+] Toggle Help

Showing Menu Help: 
  Exalogic Network Diagnostics Help
    
  Explanation of available options:
    (1) Performance (Network Health)
        o This option executes the "Exalogic_HealthNet" RDA profile which 
          can be used to collect information about the current status of the 
          network across the entire Exalogic machine. It will print details 
          about the network topology and also any errors in the InfiniBand 
          fabric. This profile can be used to troubleshoot any 	network 
          performance issues.
    
    (2) Network Cabling
        RDA will execute for the following module:
          o Exalogic_NetworkCabling
          
    (3) IB Switch
        RDA will execute for the following module with and without root
        level data:
          o Exalogic_IbSwitch
          
    (4) IB Traffic
        o This option executes the "Exalogic_IbSwitch" RDA profile which checks 
          the availability and correct functionality of all the IB switches 
          inside Exalogic rack. This profile can be run in conjunction with 
          Network Cabling" option in the menu to collect complete on all the 
          switches in Exalogic Rack and verify the correct connectivity and 
          functionality of all the cables.
    
    (5) General
        o This option executes the "Exalogic_Network" RDA profile which checks 
          for any configuration issues in the network on any given compute node. 
          This profile collects in addition to collecting the information about 
          the network topology.

COMMAND: rda_16
COMMAND TYPE: Rda
COMMAND TITLE: Exalogic Ib Switch Root
COMMAND DESCRIPTION: 
  Command Summary
    
  Runs Exalogic_IbSwitch profile at the root level to collect additional 
  information for the Exalogic_IbSwitch collection. You must have run the
  Exalogic_IbSwitch profile in normal (non-root) mode prior to this and  
  have an existing "Exalogic_IbSwitch" subdirectory under the Diagnostic 
  Assistant output directory to store this additional information.
      
  Input Requirements
    o Enter the root user password
    o Enter the switch user password
    
  Output Details
    o Shows a summary of the collection process
    o Generates a collection of files in the "Exalogic_IbSwitch" 
      subdirectory under the Diagnostic Assistant output directory that 
      contain detailed switch data
COMMAND ARGUMENT(S): 
run_profile_root=Exalogic_IbSwitch
