RACcheck - RACcheck Collection 
  RACcheck is designed to audit various important configuration settings
  within an RACcheck System - Database Servers, Storage Servers, etc.
  RACcheck MUST already be installed as this tool is NOT delivered 
  with the Diagnostic Assistant.

  Select from the following options:
    (1) Verify RacChk Configuration
    (2) Run RacChk (see Help for pre-requisites)
    (3) Run RACcheck Without Root Privilege (see Help for pre-requisites)
    (4) Collect Data After Manual DaRacChk Run

    (M)ain    (B)ack    (H)elp    (E)xit

[+] Toggle Help

Showing Menu Help: 
   Detailed Diagnostics: RacChk Help

  
  Explanation of available options:

    (1) Verify RACcheck Configuration
        o Select this option to see the pre-requisite configuration steps
          that have to be performed to run the RacChk collection from the
          Diagnostic Assistant

    (2) Run RACcheck
        o Select this option to run RacChk in non-interactive mode 
          including root privileged checks
            - root privileged checks on database servers will be executed
            - root privileged checks on storage servers will be executed
            - NO Infiniband Switch checks will be performed

          Pre-requisite configuration steps 1-5 must have been performed
          and verified before selecting this collection

    (3) Run RACcheck Without Root Privileges
        o Select this option to run RacChk in non-interactive mode without
          root privileged checks
            - No root privileged checks on database servers will be executed
            - No root privileged checks on storage servers will be executed
            - NO Infiniband Switch checks will be performed

    Pre-requisite configuration steps 1-3 must have been performed
    and verified before selecting option 4 collection.

    (4) Collect Data After Manual RACcheck Run
        o Select this option to collect data from an RACcheck run  
          performed outside of the Diagnostic Assistant (i.e. manually 
          or from another tool)
        o If you have performed option 2 or 3, you do not need to select 
          this option since the RacChk data will have been automatically 
          collected
