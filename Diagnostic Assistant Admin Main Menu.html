Oracle Diagnostic Assistant Admin Main Menu - Documentation

main - Oracle Diagnostic Assistant Admin Main Menu 
         
  The Admin Main Menu enables you to perform limited admin functions.
    
  Select one of the following options:
    (1) Manage Properties
    (2) Data Collection History
    (3) Diagnostic Assistant Version 
  
    (H)elp   (E)xit

[+] Toggle Help

Showing Menu Help: 
  Oracle Diagnostic Assistant: Admin Main Menu.   
     
  The Admin Main Menu enables you to perform limited admin functions.
    
  Explanation of available options:
    (1) Manage Properties
        o This selection provides options for viewing and managing 
          property values
        o There are currently 2 property files used by the Diagnostic Assistant
          1) da.properties which defines common properties 
          2) user.properties has limited functionality - it is 
             possible that this file is empty
        o This menu also includes an option to restore all properties back to 
          their original default values
          
        Note: some properties are not defined until prompted for the first
              time within the tool (I.e. MOS Username, CSI) or explicitly 
              set using this menu

        UNDER NO CIRCUMSTANCES MUST THE PROPERTY FILES BE MODIFIED manually. 
        Use this Manage Properties menu to perform the modifications allowed.  

    (2) Data Collection History
        o This option provides access to view summary and detailed 
          information of data collections performed
        o You can view the latest OCM collection's detail or 
          summary information.  
        o A list of a diagnostic collections executed during the
          current session is available for viewing

    (3) Diagnostic Assistant Version
        o This option displays the Diagnostic Assistant version number, 
          build number and build date
        o Also included are the versions of ADR, RDA ad OCM (if they exist)
        o This information is helpful when logging a service request for
          the Diagnostic Assistant or in debugging issues with
          Development

        o Make sure you have the latest version of the tool, by
          comparing your version with latest version documented in 
          Knowledge Article 201804.1: Diagnostic Assistant General 
          Information (https://support.oracle.com/CSP/main/article
          ?cmd=show&type=NOT&id=201804.1)

AdminProp - Manage Properties 
  
  The following options allow for viewing the properties    
  files and managing the stored properties.
      
  Select one of the following options:
    (1) View da.properties
    (2) View user.properties 
    (3) Set/Change User Configuration values
    (4) Clear User Configuration values
    (5) Allow/Disallow saving passwords
    (6) Allow/Disallow browser viewer


    (M)ain    (B)ack    (H)elp   (E)xit

[+] Toggle Help

Showing Menu Help: 
  Admin Main Menu: Manage Properties Menu Help
    
  Explanation of available options:

    (1) View da.properties
        o displays da.properties file contents
        o results will display on the screen page by page
   
    (2) View user.properties
        o displays user.properties file contents
        o results will display on the screen page by page
        
    (3) Set/Change User Configuration values
        o Permits change of property values previously set, including
          passwords (if applicable)
        o Useful in case the values were incorrectly entered when 
          initially prompted or the value has changed for any reason

    (4) Clear User Configuration values
        o Resets to the original default value for property values 
          previously set, including passwords (if applicable)
        o If no default defined, the value is removed
        o Can choose to reset a single property value or ALL
        o Does not prompt for new values; it simply removes the existing value
          and sets it to the default value, if there is one
    
    (5) Allow/Disallow saving passwords
        o By selecting this option, the 'use_wallet' property can
          be changed to TRUE (which allows passwords to be saved in a 
          secure Oracle wallet) or FALSE (which does not allow
          passwords to be saved, and so the user is prompted each time).
        o This property is set to FALSE by default
              
    (6) Allow/Disallow browser viewer
        o By selecting this option, the 'browser_viewer_spawned' property can
          be changed to TRUE.
        o This property is set to FALSE by default
 
    Note: Oracle wallet is used to secure passwords

ResetProp - Reset User Configuration 
  
  The following options allow for changing property values.
      
  Select one of the following options:
    (1) Reset a specific property value
    (2) Reset a saved password
    (3) Reset ALL Stored Properties and Passwords

    (M)ain    (B)ack    (H)elp   (E)xit

[+] Toggle Help

Showing Menu Help: 
  Admin Main Menu: Reset Properties Menu Help
    
  Explanation of available options:

    (1) Reset a specific property value
        o Select from the list of available properties displayed
        o Resets a single value stored in da.properties or
          user.properties to the default, or NO VALUE if there is not one
          
    (2) Reset a saved password
        o Select from the list of available passwords displayed
        o Removes the value stored for the password chosen
        
    (3) Reset ALL Stored Properties and Passwords
        o Removes the values from all properties and passwords
        o Sets those that have a default to that value, or NO VALUE if there 
          is not one

AdminFile - Data Collection History 
   
  Data Collection History allows the viewing of OCM summary 
  and detail information as well as other diagnostic files.
    
  Select one of the following options:
    (1) Latest OCM Collection Summary
    (2) Latest OCM Collection Detail
    (3) RDA Collections Summary
    (4) RDA Collections Detail
    (5) ADR Collections Summary
    (6) ADR Collections Detail

    (M)ain    (B)ack    (H)elp   (E)xit

[+] Toggle Help

Showing Menu Help: 
   
  Admin Main Menu: Data Collection History Menu Help

  Explanation of available options:

    (1) Latest OCM Collections Summary
        o Retrieves the latest configuration summary data and provides
          immediate viewing of this data

    (2) Latest OCM Collections Detail
        o  Retrieves the latest configuration summary data and provides
           immediate viewing of this data
          
    (3) RDA Collections Summary
        o List of all RDA collections, including what and when
  
    (4) RDA Collections Detail
        o List of all the files created for each collection listed 
          in the summary
        o Output will be extensive, but a paging menu will be 
          available for scrolling
            
    (5) ADR Collections Summary
        o List of all ADR incidents
   
    (6) ADR Collections Detail
        o Shows incident detail
        o Future: additional detail is planned
